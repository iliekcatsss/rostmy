  Rostmy
</h1>
Rostmy es un creador de notas, pensado para la eficiente gestión de archivos de texto.

# Propiedades

- Editor markdown con preview en tiempo real
- Organización por carpetas anidadas
- Multi-usuario con datos separados
- Accesible desde cualquier dispositivo

---

Para probar el markdown, copia y pega lo siguiente en un archivo:
```
# Header 1
## Header 2
### Header 3

Párrafo de una línea

> Hola mundo!

Párrafo\
Multi-línea

1. **negritas** __negritas__
2. _cursiva_ *cursiva*
2. `código` y ~tachado~
   - 1
     - 2
       - 3

Link: https://example.com/

[Link (más bonito)](https://example.com/)

![](https://rostmy.vercel.app/favicon.ico)

------

- [ ] casilla vacía
- [x] casilla marcada

| Header 1 | Header 2 |
| --- | --- |
| Contenido 1 | Contenido 2 |


![hola](https://i.imgur.com/iPd2q4R.png)
```
